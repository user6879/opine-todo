## TODO app written in Deno using Opine.

It's the first time I'm creating something using Deno.
<br>

### Installation

Clone this repository

```bash
git clone https://github.com/akumaru69/opine-todo
```

<br>

Running the code:<br>

```bash
deno run -A core.ts
```

Or you can use deno tasks

```bash
deno task dev
```

or

```bash
deno task prod
```
